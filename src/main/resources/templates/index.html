Eigenvectors and Eigenvalues of Symmetric Matrices (WIP)

Background: This API calculates the eigenvectors and eigenvalues (together often called "diagonalization) of symmetric real matrices. Many calculations involving such a matrix can only be done easily - if at all - after it has been diagonalized.
Instructions: After the url above you should type / followed by a comma-separated list of the column vectors which do not include elements in the lower-triangular portion of the matrix. Each column vector is a parenthesis-wrapped comma-separated list of numbers. For instance the symmetric matrix depiced below is specified by
...heroku.com/(1),(2,3),(4,5,6).
The eigenvalues for this matrix are approximately -0.057, -1.507, and 11.564.
